# easyUtil_poped
Modal Dialogue Box by  JavaScript
<h4>简介</h4>
		<h5>easyUtil.poped是简单的模态弹出窗口，拥有提示框，警告框，对话框以及弹出窗口功能，有两种颜色样式可以选择，也可以自定义样式，修改css文件对应类即可。</h5>
		<h5>poped有两个文件：一个是用于模块化引入（以module命名），支持seajs；一个是普通的js文件（以normal命名），可根据实际开发情况分别引入。</h5>
		<h4>API:</h4>
		<h5>入口</h5>
		<ul>
			<li>1.模块化导入的入口不用介绍了，会用seajs的直接导入，自定义引用如下方法名即可</li>
			<li>2.普通js，调用名称为easyPoped</li>
		</ul>
